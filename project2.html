CHATBOT Using AWS

Developed using AWS cloud (free-tire)

AWS Chatbot is an interactive agent that makes it easy to monitor and interact with your AWS resources and Amazon Chime chat rooms. With AWS Chatbot you can receive alerts, run commands to return diagnostic information, invoke AWS Lambda functions, and create AWS support cases.


The CHATBOT can be deployed on any application specified by the AWS.
