WEB PORTFOLIO

Developed using "HTML and CSS" using the IDE "Visual Studio Code"

I recently designed my own portfolio website. It was a challenging but enjoyable experience that I really learned a lot from.I would like to thank Brad Traversey for demonstrating this wonderful template. For creating this website I have used "VS Code", "Node JS" and "Git bash". I have used "SCSS" as an extended stylesheet language as it has the functionality that did not exists in "CSS". Since HTML do not understand SCSS language the code written in SCSS has been converted to CSS language. For making website go online I have used GITHUB.
