EVENT MANAGEMENT SYSTEM

Developed using languages "java with xampp" and "c++ with file structure" respectively.

Event Management System is developed in Java programming language.In the existing system, there is a lot of pater work; it is very time consuming and uneconomical as most of the works include manual processing. The records are difficult to store in manual system, and it requires more manual labor work.
In the proposed event management system, everything is application-based. This will immediately reduce the manual processing, thereby increasing the speed of event management process. There are various functions and modules in the system to perform various features.

The modules of the project are:
1. Administrator/ Admin module
2. User module
3. Event registration module
4. Event cancellation module
5. Booking module
